# Coffee Dynamics

#### Mug Dimensions

##
 - [0.035m] inner radius of mug (R2)
 - [0.0105m] frother froth disc radius (R1)
 - [0.04m] how deep liquid sits when i froth from BOTTOM UP (H)
 - [0.0425m] outer radius of mug
 - [0.105m] height of mug
 - [0.0245m] gap width d = R2 - R1
 - [0.30] gap ratio n = R1/R2
 - [1,257 rad/s] angular velocity (omega1)

## Frother
- https://www.amazon.com/CIRCLE-JOY-Rechargeable-Cappuccino-Chocolate/dp/B0FT3VTD72?th=1 
- 10W motor. 3 Speed Setting (i froth with MID): Mid ~12,000 RPM
- 12,000 rev/min x 2pi rad/rev x 1 min/60 s = 1,257 rad/s

## Reynolds Numbers
- Re = omega1*R1*d / v = 1247 * 0.0105 * 0.0245 / 3.3x10^-7 = 981000
- Re = 981,000 EXTREMELY VIOLENT FLOW
